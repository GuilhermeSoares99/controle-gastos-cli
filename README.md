# 💰 Controle de Gastos CLI

![CI](https://img.shields.io/badge/build-passing-brightgreen)

## 📌 Problema
Muitas pessoas possuem dificuldade em controlar seus gastos diários, o que pode levar a endividamento e falta de planejamento financeiro.

## 💡 Solução
Aplicação em linha de comando (CLI) que permite registrar, listar e calcular gastos, ajudando no controle financeiro pessoal de forma simples e acessível.

## 👥 Público-alvo
- Estudantes
- Pessoas com dificuldade em organização financeira
- Qualquer pessoa que deseja controlar gastos rapidamente

## ⚙️ Funcionalidades
- Adicionar gasto
- Listar gastos
- Calcular total
- Persistência em arquivo JSON

## 🛠️ Tecnologias
- Python
- Pytest
- Flake8
- GitHub Actions

## 🚀 Instalação
```bash
pip install -r requirements.txt
```

## ▶️ Execução
```bash
python src/main.py
```

## 🧪 Testes
```bash
pytest
```

## 🔍 Lint
```bash
flake8 src
```
